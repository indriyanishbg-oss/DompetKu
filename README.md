# 💳 DompetKu - Expense Tracker App

Aplikasi **DompetKu** adalah aplikasi pencatatan keuangan sederhana berbasis React Native (Expo).  
Aplikasi ini digunakan untuk mencatat pemasukan dan pengeluaran harian serta menghitung saldo otomatis.

---

## 🚀 Fitur Aplikasi

- 💰 Menampilkan saldo otomatis
- ➕ Menambah transaksi pemasukan
- ➖ Menambah transaksi pengeluaran
- 📊 Menampilkan total pemasukan & pengeluaran
- 🧾 Riwayat transaksi menggunakan FlatList
- 🎨 UI modern berbasis card design
- 🗑️ Hapus transaksi
- 🎯 Warna otomatis (Hijau = masuk, Merah = keluar)
